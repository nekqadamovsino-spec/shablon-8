# Андрей & Анастасия — Story Invitation

Мобильный сайт-приглашение в стиле Instagram Stories / cinematic editorial.

## Загрузка на GitHub Pages

1. Создай Public репозиторий.
2. Загрузи файлы из этой папки: `index.html`, `style.css`, `script.js`, папку `images`.
3. Открой Settings → Pages.
4. Source: Deploy from a branch.
5. Branch: `main`, folder: `/root`.
6. Save.
